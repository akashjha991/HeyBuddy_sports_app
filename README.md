# HeyBuddy Sports App (Frontend Scaffold)

This repository now contains a working Next.js App Router frontend scaffold for a mobile-first sports room discovery platform.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included modules
- Auth pages (`/auth/login`, `/auth/signup`, `/auth/forgot-password`)
- Dashboard (`/dashboard`)
- Create room (`/rooms/create`)
- Room detail (`/rooms/[id]`)
- Profile (`/profile`)
- Shared UI components and API/store scaffolding
